# express-REST

## currently handles authentication of users for a react app, by enabling the use of login, signup and signout.

## Data Model

### User

|  field   |  Type  |
| -------- | ------ |
| Id       | String |
| Name     | String |
| Email    | String |
| Password | String |


## Available Routes

* /users
* /login

## Available Methods

| Http Method | Supported   |
| ----------- | :---------: |
|    GET      |      ✔️      |
|    POST     |      ✔️      |
|    DELETE   |      ✔️      |
|    PATCH    |      ✔️      |
|    PUT      |      ❌     |
